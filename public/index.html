⚡

Zero Codegen

Other tools generate static TypeScript from your spec. mcpwrap wraps your API at runtime — no files to generate, maintain, or commit.

🔄

Always Fresh

Update the spec, restart the server, done. No regeneration step. Works with remote specs that change over time.

🔐

Auth Passthrough

Bearer tokens, API key headers, query param auth. Pass --auth "Bearer $TOKEN" and it flows through to every request.

Claude Desktop Config

Add any API to Claude in 30 seconds. No plugins, no custom code.

{
  "mcpServers": {
    "my-api": {
      "command": "npx",
      "args": ["mcpwrap", "--spec", "https://api.example.com/openapi.json"],
      "env": { "API_KEY": "your-key-here" }
    }
  }
}
